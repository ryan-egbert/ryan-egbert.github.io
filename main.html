We created this website to show everyone how literally awesome languages are. The map below allows you to see where language are spoken in the world. You can select as many languages as you like using our “Multiselect Languages” button. If you would like to see the history and evolution of a language from a family, you can use the “Language Relationship” button. This will show you the connections that just one language has with other family members. The slider tool allows you to scale the lineage. The scale that we use is a 1-10 scale. The default option is set at 1 which will show you just the language you selected. Each increment will show you direct relationships that language has with its family.
